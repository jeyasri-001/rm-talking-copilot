# Playbook — Fixed deposit maturity: reinvestment options

## Context
Bank FDs are offering 6.75-7.25% for 1-3y tenors. Post-tax for a 30% slab client this is **4.7-5.0%** — barely above inflation.

## Alternatives we recommend
| Option | Indicative yield | Tax treatment | Lock-in | Best for |
|--------|-----------------:|---------------|---------|----------|
| Arbitrage fund | 6.0-6.5% | Equity (LTCG @ 12.5% > 1y) | None | 3-12m parking |
| Target maturity debt ETF (2028/2031) | 7.0-7.2% | Debt (slab) | Maturity | Matched liability |
| Corporate bond MF (AAA) | 7.2-7.4% | Debt (slab) | None | 2-3y horizon |
| Gilt + dynamic bond combo | 7.0-7.3% + capital gain | Debt (slab) | None | Rate-cut view |
| Sovereign Gold Bond | 2.5% coupon + gold price | LTCG-free if held to maturity | 8y (exit after 5) | Long-term diversifier |

## Process
1. Confirm liquidity need and horizon with client.
2. Map to 2-3 options from the table; never pitch a single product.
3. Produce a one-page comparison sheet (post-tax yield, horizon, risk) before the call.
4. Document in CRM why a specific choice was made — suitability trail.

## Compliance
- **Never represent any of these as "FD replacements"** — they are not capital-protected.
- For conservative clients (risk score ≤ 3): default to TMF/BBF ETFs, not dynamic bond funds.
- If the FD is jointly held, the reinvestment must be in the same ownership structure or documented as a gift.

## Talking points
1. "Your post-tax FD return is around 5%. We can target 6.5-7% post-tax with comparable risk using target-maturity funds."
2. "We are not chasing yield — we are keeping duration and credit quality close to what you already accept."
3. "If rates fall as we expect, a 3-5% capital gain on top is a bonus, not the base case."
